The Purerl Cookbook
###################

This is a collection of code examples/explanations of most things you might want to do in the Purerl world.

If you feel anything is missing, please head over to Github and open an issue or pull request and we'll try and get it sorted.


Useful things to look at
------------------------

* `Purescript <https://www.purescript.org>`_ - the actual language we're using here
* `Purerl <https://github.com/purerl/purescript>`_ - the backend for the Purescript compiler
* `Pursuit <http://pursuit.purerl.fun>`_ - module/function search for or Purerl 
* `Pinto <https://github.com/id3as/purescript-erl-pinto>`_  - an opinionated wrapper for OTP in Purescript
* `Stetson <https://github.com/id3as/purescript-erl-stetson>`_ - an opinionated wrapper for Cowboy in Purescript
* `Spago <https://github.com/purescript/spago>`_ - tool for managing packages in Purescript
* `Dhall <https://dhall-lang.org/>`_ - configuration language used for packaging in Purescript
